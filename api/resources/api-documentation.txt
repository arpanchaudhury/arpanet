                                    API Documentation
                                    -----------------


Routes                                                      Description
======                                                      ===========

GET     /resume                                             Get resume as 'application/pdf'

GET     /write-ups                                          Get paginated list of blog posts filtered by topics <Query Parameters - pageStart, pageLength, topics>

GET     /write-ups/topics                                   Get all topics from blog post collection

GET     /images/photography                                 Get paginated details of photographs filtered by tags <Query Parameters - pageStart, pageLength, tags>

GET     /images/photography/:imageId                        Get a photograph as 'image/png' when valid imageId is provided

GET     /images/public-image/:imageId                       Get a public image as 'image/jpeg' when valid imageId is provided

GET     /search/write-ups                                   Get set of paginated search results for blog posts

GET     /search/photographs                                 Get set of paginated search results for photographs
